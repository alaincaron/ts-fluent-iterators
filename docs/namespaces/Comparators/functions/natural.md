# Function: natural()

> **natural**\<`A`\>(`a1`, `a2`): `-1` \| `0` \| `1`

Natural comparator.

## Type Parameters

• **A**

The type of elements to be compared

## Parameters

• **a1**: `A`

The left-hand size argument

• **a2**: `A`

The right-hand size argument

## Returns

`-1` \| `0` \| `1`

- -1 if `a1 < a2`,
- 1 if `a1 > a2`,
- 0 otherwise.
